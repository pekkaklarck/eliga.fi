Welcome!

Hi, I'm Pekka Klärck, an independent consultant and the author of Robot Framework. I'm based on Helsinki, Finland.

Robot Framework is a generic open source framework for test automation, behavior driven development (BDD), and robotic process automation (RPA). It is used widely around the world.

I develop Robot Framework further with sponsoring by Robot Framework Foundation. In addition to that I organize both public and private training courses and provide also other services related to the framework. Please contact me directly if you are interested!
